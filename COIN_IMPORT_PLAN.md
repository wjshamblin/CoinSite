## Image Acquisition Strategy

### Challenge
The collectivecoin.com site is JavaScript-rendered and cannot be scraped directly for images.

### Options

1. **Manual Download** (Recommended)
   - User logs into collectivecoin.com
   - Downloads images for each coin
   - Places in `/public/images/coins/` with naming convention:
     ```
     [slug]-1.jpg (primary/obverse)
     [slug]-2.jpg (reverse)
     [slug]-3.jpg (edge/detail)
     ```

2. **ColleOnline Archive**
   - Site is shutting down but may have cached images
   - User could request data export

3. **CoinTalk Forum**
   - Many coins have images embedded in forum posts
   - Would need manual extraction

4. **Placeholder Strategy**
   - Import coins with placeholder images initially
   - User adds real images over time via admin interface
